## Implementation :

Implementation  To use this project, you need the behave library

```bash
pip install behave
```
**Runing for IDE**</br>
for run Click ctrl + F5 or Click Run Bottom 

**Runing for Terminal**</br>
for example :
```bash
python3 main.py
```

**Runing for command port**</br>
for example :
```bash
pyhton main.py
```
